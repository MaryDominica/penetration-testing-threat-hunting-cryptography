# Google Dorking in Penetration Testing

## Definition
Google Dorking refers to the use of advanced search techniques that uses specialized queries to uncover information that is not
publicly accessible through a standard Google search.

## Ethical Use
Google Dorking must only be used:
- In authorised assessments
- Maintaining good intentions
- For educational or defensive purposes
- Within defined scope

## Google Dork Commands
- After
- Allinurl
- Allintitle
- AROUND(X)
- Before
- Define
- Filetype
- Inanchor
- Info
- Intext
- Inur
- Link
- Map
- Numrange
- Phonebook
- Related

## Combining Search Operators
Search operators can be combined using AND and OR to refine results and identify exposed
information more efficiently. 

## Risks of Exposed Information
- Data leakage
- Sensitive file exposure
- Misconfigured systems

## Defensive Perspective
Organisations should:
- Monitor publicly available data
- Secure sensitive files
- Implement proper access controls

## Key Takeaway
Google Dorking highlights the importance of managing publicly exposed
information as part of security hygiene.
