# Planning Phase
The planning phase defines how the penetration test will be conducted and includes:
- Penetration testing scope  
  - Five core elements that define what systems, assets, and techniques are permitted
- Rules of engagement  
  - Eight key concepts that establish legal, ethical, and operational boundaries

Together, the scope and rules of engagement ensure penetration testing is controlled,
authorised, and aligned with organisational objectives.


Discovery Phase (Reconnaissance)
The discovery phase focuses on information gathering and includes:
- Passive reconnaissance
  - Collecting publicly available information without directly interacting with target systems
- Active reconnaissance
  - Direct interaction with systems in a controlled and authorised environment

This module also covered:
- Ethical use of Google Dorking in penetration testing
- Purpose of Google Dorking for identifying exposed information
- Use of search operators, wildcards, and special characters
- Combining Google Dork commands responsibly for security assessment
